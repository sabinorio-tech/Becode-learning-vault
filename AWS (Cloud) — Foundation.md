
## 🌐 1. Definition

Amazon Web Services: 

> AWS = cloud platform to rent computing power, storage, and infrastructure

---
## 🧱 2. Core services

|Type|Service|Use|
|---|---|---|
|Servers|EC2|Run apps|
|Storage|S3|Store data|
|Security|IAM|Manage access|
|Functions|Lambda|Run code automatically|
|Databases|RDS|Store structured data|

---
## 3. 🔁 System flow

	User → API → Compute → Storage/DB
