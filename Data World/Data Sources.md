---
## 🧠 2. Main Types

### 🟦 A) Databases

- structured data
- stored and managed systems
- queried using SQL or ORM

-> [[Databases]]

### 🟩 B) APIs

- structured data (JSON) 
- data provided by external services
- accessed via HTTP requests

-> [[API]]

### 🟨 C) Web Scraping

- HTML pages 
- requires parsing
- used when no API is available

-> [[Web Scraping]]

### 🟪 D) Files

- CSV 
- JSON
- Excel
- local or static data

## ⚖️ 3. Comparison

| Source    | Difficulty  | Structure    |
| --------- | ----------- | ------------ |
| Databases | Medium      | Structured   |
| API       | Easy        | Structured   |
| Scraping  | Medium/Hard | Unstructured |
